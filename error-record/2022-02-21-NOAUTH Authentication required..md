# 报错信息：
 - 展示了连接redis没有权限
 - 经过查询资料得知，连接redis服务器没有配置密码

# 解决办法：
- 添加密码

# 代码如下：
```
@Test
    public void test01(){
        Jedis jedis = new Jedis("101.200.178.253", 6379);
       jedis.auth("redis密码");
        jedis.set("k12","v12");
        String k12 = jedis.get("k12");
        // v12
        System.out.println(k12);
    }
```
